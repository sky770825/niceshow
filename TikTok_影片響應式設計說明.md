## 🔧 技術細節

### CSS 特性使用

**Grid 布局（電腦/平板）:**
```css
display: grid;
grid-template-columns: repeat(4, 1fr);
gap: 1.5rem;
```

**Flexbox 滑軌（手機）:**
```css
display: flex;
overflow-x: auto;
scroll-snap-type: x mandatory;
```

**Scroll Snap（滑動吸附）:**
```css
scroll-snap-align: center;
scroll-snap-stop: always;
```

**平滑滾動:**
```css
-webkit-overflow-scrolling: touch;
```

## 🔄 新增影片流程

### 步驟 1: 複製影片代碼
從 TikTok 取得嵌入代碼：
```html
<blockquote class="tiktok-embed" cite="..." data-video-id="..." >
    ...
</blockquote>
```

### 步驟 2: 包裝在容器中
```html
<div class="tiktok-item" style="border-radius: 15px; overflow: hidden; box-shadow: 0 4px 20px rgba(0, 0, 0, 0.1);">
    <div class="tiktok-container" style="height: 680px; overflow: hidden;">
        <!-- TikTok 嵌入代碼 -->
    </div>
</div>
```

### 步驟 3: 加到網格中
放在 `<div class="tiktok-grid">` 裡面

### 步驟 4: 更新指示器
在 `<div class="tiktok-dots">` 中新增：
```html
<span class="tiktok-dot" onclick="scrollToVideo(X)"></span>
```
（X 是影片索引，從 0 開始）

## 📞 需要調整？

### 修改影片高度
編輯 `style.css` 中對應的螢幕尺寸區塊：
```css
@media (min-width: 1920px) {
    .tiktok-container {
        height: 580px !important;  /* 修改這裡 */
    }
}
```

### 修改欄數
編輯 `style.css`:
```css
grid-template-columns: repeat(4, 1fr);  /* 4 改成想要的欄數 */
```

### 修改間距
編輯 `style.css`:
```css
gap: 1.5rem;  /* 修改間距 */
```
