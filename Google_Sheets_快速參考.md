## ✅ 必填欄位格式

### 店名（B欄）
```
✅ 饞了找我
✅ 宇宙雞蛋糕
```

### 預約場地（D欄）
```
✅ 漢堡大亨 四維路70號(週一~週六)
✅ 四維路60號
✅ 台東放山豬_楊梅_四維路216號
```

### 預約日期（E欄）
```
✅ 10月21日(星期一)
✅ 10月21日（星期一）
❌ 10/21              （缺少月日字）
❌ 10月21日 星期一    （缺少括號）
❌ 10月21日(週一)     （應該是「星期一」）
```

### 己排（F欄）
```
✅ 己排班            （完全一致，包含「班」字）
❌ 己排              （缺少「班」字）
❌ 待排              （不會顯示）
❌ （空白）           （不會顯示）
```

   ```

   ```


## 🐛 快速除錯

### 問題：連接失敗
```
1. 確認 Google Sheets 設為「知道連結的任何人」可檢視
2. 清除瀏覽器快取
3. 檢查網路連線
```

### 問題：餐車沒有顯示
```
1. 確認「己排」欄位填寫「己排班」（注意有「班」字）
2. 確認日期格式：10月21日(星期一)
3. 確認地址包含支援的四維路地址
```

### 問題：日期錯誤
```
正確格式：10月21日(星期一) 或 10月21日（星期一）
星期要用：星期一、星期二... （不是週一、週二...）
```

## 📝 新增餐車流程

1. 在 Google Sheets 中新增一行
2. 填寫必要欄位：
   - **您的店名**：餐車名稱
   - **預約場地**：包含四維路地址
   - **預約日期**：`10月21日(星期一)`
   - **己排**：`己排班`
3. 重新整理網頁
4. 確認餐車已顯示

## 💡 小技巧

1. **批量更新**：在 Google Sheets 中可以一次填寫多天的餐車資料
2. **複製貼上**：可以複製前一週的資料，只需修改日期
3. **快取清除**：如果更新後沒有變化，按 `Ctrl + F5` 強制重新整理
4. **多人協作**：可以邀請其他人一起編輯 Google Sheets
5. **自動備份**：Google Sheets 會自動儲存版本歷史記錄
