🔑 GitHub Token 設定

設定 GitHub Token 後，您的餐車資料就能自動同步到 GitHub 了！

⚠️ 注意：您的 Token 驗證失敗 (401 錯誤)，請重新設定一個有效的 Token。

📋 如何取得新的 GitHub Token：

  1. 前往 GitHub Token 設定頁面
  2. 點擊「Generate new token」→「Generate new token (classic)」
  3. 填寫 Token 名稱：餐開月行程表同步
  4. 選擇過期時間：No expiration
  5. 勾選權限：repo（完整倉庫存取權限）
  6. 點擊「Generate token」
  7. 複製產生的 Token（只會顯示一次！）
請輸入您的 GitHub Personal Access Token

✅ 設定完成後：

  • 編輯餐車資料會自動同步到 GitHub
  • F5 重新載入會載入最新的資料
  • 管理後台的「儲存並同步」功能將正常運作

🔄 完成設定後：

請回到 管理後台 測試同步功能。
