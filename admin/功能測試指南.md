# 功能測試指南

## 🧪 測試前準備

1. **確認 Supabase 設定**
   - 已建立 `.env` 檔案
   - 已填入 Supabase URL 和 Anon Key
   - 已在 Supabase 中建立 `niceshow` bucket
   - Bucket 已設為公開（Public）

2. **啟動開發伺服器**
   ```bash
   npm run dev
   ```

## 📋 測試項目

### 1. 登入功能測試
- [ ] 訪問 `http://localhost:3000`（或顯示的端口）
- [ ] 輸入錯誤密碼，應該顯示 Toast 錯誤通知
- [ ] 輸入正確密碼（admin123 或 foodtruck2024）
- [ ] 應該顯示 Toast 成功通知並跳轉到儀表板

### 2. Toast 通知系統測試
