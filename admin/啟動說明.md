# 後台管理系統啟動說明

## 快速開始

### 1. 安裝 Node.js

確保您的電腦已安裝 Node.js（建議版本 18 或以上）

檢查方式：
```bash
node --version
npm --version
```

如果沒有安裝，請到 [nodejs.org](https://nodejs.org/) 下載安裝。

### 2. 安裝專案依賴

在 `admin` 資料夾中執行：

```bash
cd admin
npm install
```

這會安裝所有需要的套件（約需 1-2 分鐘）

### 3. 啟動開發伺服器

```bash
npm run dev
```

系統會自動在瀏覽器開啟 `http://localhost:3001`

### 4. 登入系統

使用以下任一密碼登入：
- `admin123`
- `foodtruck2024`

## 功能說明

### 📊 儀表板
- 查看整體統計資料
- 快速進入各管理功能

### 📅 餐車行程管理
- 從 Google Sheets 讀取行程資料
- 點擊「開啟 Google Sheets」直接編輯
- 編輯後點擊「重新載入」更新

### 🖼️ 圖片管理
- 管理跑碼燈圖片
- 可編輯圖片資訊、調整順序
- 匯出 JSON 後手動更新 `data.json`

### 🎬 影片管理
- 管理 TikTok 影片
- 自動從 URL 提取影片 ID
- 編輯後需手動更新 `index.html`

### 🛠️ 工具推薦管理
- 管理實用工具、相關推薦、廠商進駐
- 三個分類標籤切換
- 編輯後需手動更新 `index.html`

### 🏥 醫療資訊管理
- 管理五個類別的醫療機構
- 完整編輯機構資訊
- 編輯後需手動更新 `index.html`

## 注意事項

⚠️ **重要提醒：**

由於瀏覽器安全限制，系統無法直接寫入檔案。編輯完成後需要：

1. **圖片管理**：點擊「匯出 JSON」，下載後替換專案中的 `data.json`
2. **其他管理**：手動複製編輯後的內容到 `index.html` 對應區塊

## 停止伺服器

在終端機中按 `Ctrl + C` 即可停止開發伺服器。

## 問題排除

### 無法安裝依賴
- 檢查網路連線
- 嘗試清除快取：`npm cache clean --force`
- 刪除 `node_modules` 資料夾後重新安裝

### 無法啟動
- 確認已安裝 Node.js
- 確認在正確的資料夾中執行命令
- 檢查 3001 埠是否被占用

### 無法載入資料
- 檢查 Google Sheets 是否設為公開
- 確認網路連線正常
- 檢查瀏覽器控制台是否有錯誤訊息

## 技術支援

如有問題，請檢查：
1. 瀏覽器控制台（F12）的錯誤訊息
2. 終端機的錯誤訊息
3. 確認所有檔案都在正確位置
