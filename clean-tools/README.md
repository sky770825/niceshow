# 🛠️ 實用工具集 - 乾淨版本

這是一個經過清理和優化的實用工具集合，包含4個穩定工作的工具。

## 📋 工具列表

### 1. 🧮 紗窗報價計算器
- **功能**: 專業的紗窗報價計算工具
- **特色**: 
  - 精確計算平紗、捲紗等多種類型
  - 內建陽台工程計算功能
  - 自動計算材料費、工本費、稅金
  - 響應式設計，支援各種設備
- **文件**: `screens-calculator.html`

### 2. 📈 虛擬幣交易輔助系統
- **功能**: 專業的 BTC/ETH 交易輔助工具
- **特色**:
  - 即時監控 K 線突破/跌破信號
  - 多週期技術分析支援
  - 價量驗證與信號確認
  - 進場點、停損與目標位建議
- **文件**: `crypto-trading.html`

### 3. 🧘‍♀️ 身心靈瘦身小測驗
- **功能**: 專業的身心靈瘦身測驗工具
- **特色**:
  - 塔羅星空風格設計
  - 身心靈瘦身傾向分析
  - 個性化健康建議
  - 美麗的星空背景效果
- **文件**: `mind-body-soul-quiz.html`

### 4. 🚀 完全自動化上傳系統
- **功能**: 一鍵上傳 HTML 文件到 GitHub Pages
- **特色**:
  - 拖拽上傳多個文件
  - 自動部署到 GitHub Pages
  - 即時進度顯示
  - 支援多種文件格式
- **文件**: `auto-upload.html`

## 🚀 使用方式

1. **本地使用**: 直接在瀏覽器中打開 `index.html`
2. **部署到 GitHub Pages**: 將整個 `clean-tools` 資料夾上傳到 GitHub 並啟用 Pages

## 📁 文件結構

```
clean-tools/
├── index.html              # 主頁面
├── screens-calculator.html  # 紗窗計算器
├── crypto-trading.html     # 虛擬幣交易輔助
├── mind-body-soul-quiz.html # 身心靈瘦身測驗
├── auto-upload.html        # 自動化上傳系統
└── README.md               # 說明文件
```
